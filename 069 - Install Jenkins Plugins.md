```sql
Jenkins Plugin Installation Steps
Log in to the Jenkins UI using your credentials.
Go to Manage Jenkins from the left-hand menu.
Open Plugins under System Configuration.
Click on Update Plugins.
Select the plugins that need to be updated.
Proceed with the update process.
When prompted, choose to restart Jenkins after updates are completed (if no jobs are running).
Again go to Manage Jenkins → Plugins.
Open the Available Plugins tab.
Search for Git and select it.
Search for GitLab and select it.
Click on Install to start plugin installation.
Wait for the installation to complete.
At the end, select the option to restart Jenkins to apply changes.
Check in installed plugins and see if all there.
```
